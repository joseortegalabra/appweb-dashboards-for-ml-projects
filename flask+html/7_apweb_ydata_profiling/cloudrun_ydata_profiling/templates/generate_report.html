Instrucciones

- Subir archivo con las extensiones "csv", "excel", "pkl" con los datos a los que se le desean realizar un perfilamiento a los datos. Importante: cada tag debe estar en una columna // La primera columna corresponde a las fechas // El resto de columnas son los tags

- Se puede elegir dos opciones: Realizar el perfilamiento a un conjunto de datos. Realizar el perfilamiento a dos conjuntos de datos comparándolos (por ejemplo: comparar train vs test, comparar datos antes de limpieza vs después de limpieza)

- El perfilamiento de los datos (ydata-profiling) tiene un alto costo computacional para realizar algunos análisis (correlaciones, interacciones, missing values, valores duplicados). Por lo cual existe la opción de realizar un perfilamiento minimalista (minimal) el cual únicamente analiza las variables (percentiles, histogramas, mínimos, máximos) y omite los análisis con alto costo computacional (correlaciones, interacciones, etc). Este análisis minimal sirve como punto de partida para explorar datasets de gran tamaño

- Esta web está configurado para realizar una análisis minimal si el tamaño del dataset subido es superior a 5000 observaciones

- Luego de subir el/los conjuntos de datos se abrirá una nueva pestaña en blanco que cargará mientras se genera el reporte de profiling. Luego de terminar de generarse el reporte se cerrará la pestaña en blanco y se descargará de forma automática el reporte
